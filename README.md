# Flattice
Flat and simple design make your desktop clean and beautiful!

This theme is based on Flatts_09022014 (nale12's artworks)

GTK2 ENGINES REQUIREMENT
- GTK2 engine Murrine 
- GTK2 pixbuf engine or the gtk(2)-engines package.
